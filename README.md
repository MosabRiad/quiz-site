# 🎓 موقع الاختبارات - Quiz Site

منصة اختبارات تفاعلية مع لوحة إدارة متقدمة لتوليد الأسئلة تلقائياً من ملفات PowerPoint باستخدام Google Gemini AI.

## ✨ المميزات الرئيسية

### 🎯 للطلاب
- ✅ واجهة اختبار سهلة وودية
- ✅ شريط تقدم يوضح المسار
- ✅ مؤقت عكسي لإدارة الوقت
- ✅ التنقل بين الأسئلة بسهولة
- ✅ عرض النتائج الفورية
- ✅ دعم كامل للغة العربية

### 👨‍💼 للمعلمين/الإدارة
- 📤 رفع ملفات PowerPoint مباشرة
- 🤖 توليد أسئلة تلقائية باستخدام Gemini AI
- 🎨 اختيار نوع الأسئلة (اختيار من متعدد، صح/خطأ، إجابة قصيرة)
- 📊 إدارة مجموعات الأسئلة
- 📈 تتبع نتائج الطلاب
- 💾 حفظ البيانات محلياً (localStorage)

## 🚀 البدء السريع

### المتطلبات
- متصفح حديث يدعم JavaScript
- اتصال إنترنت
- مفتاح Google Gemini API (مجاني)

### الخطوات

#### 1. الحصول على مفتاح Gemini API

1. اذهب إلى [Google AI Studio](https://makersuite.google.com/app/apikey)
2. انقر على "Create API Key"
3. اختر "Create API key in new project"
4. انسخ المفتاح

#### 2. استخدام لوحة الإدارة

```
1. افتح ملف admin.html
2. أدخل مفتاح Gemini API الخاص بك
3. اختر ملف PowerPoint من جهازك
4. اختر إعدادات الأسئلة (عدد الأسئلة، النوع، الصعوبة)
5. انقر على "توليد الأسئلة"
6. راجع المعاينة وانقر "حفظ الأسئلة"
7. انقر "تفعيل" لاستخدام المجموعة
```

#### 3. استخدام الاختبار

```
1. افتح ملف index.html
2. أدخل اسمك
3. أجب على الأسئلة
4. انقر "إرسال الإجابات"
5. اعرض نتائجك
```

## 🔧 كيفية العمل

### معالجة ملفات PowerPoint

```javascript
// استخدام JSZip لفك ضغط الملف
// استخراج النصوص من XML
// تمرير النصوص إلى Gemini API
```

### توليد الأسئلة

```javascript
// الخطوة 1: استخراج النصوص من PowerPoint
// الخطوة 2: إرسال النصوص مع prompt مخصص إلى Gemini
// الخطوة 3: معالجة الرد وتحويله إلى أسئلة
// الخطوة 4: عرض المعاينة للمراجعة
// الخطوة 5: حفظ في localStorage
```

### تخزين البيانات

تستخدم المنصة `localStorage` لتخزين:
- **الأسئلة المحفوظة** - `quizQuestions`
- **نتائج الطلاب** - `studentResults`
- **الأسئلة الحالية** - `currentQuiz`
- **مفتاح Gemini API** - `geminiKey` (مشفر)

## 🎨 التخصيص

### تغيير الألوان

عدّل متغيرات CSS في `style.css` و `admin.css`:

```css
:root {
    --primary: #667eea;      /* الأزرق الأساسي */
    --secondary: #764ba2;    /* البنفسجي */
    --success: #48bb78;      /* الأخضر */
    --danger: #f56565;       /* الأحمر */
}
```

### تعديل عدد الأسئلة

في `admin.html` غيّر القيمة الافتراضية:

```html
<input type="number" id="questionCount" value="5" min="1" max="50">
```

### تغيير وقت الاختبار

في `script.js` غيّر هذا السطر:

```javascript
let timeRemaining = 30 * 60; // 30 دقيقة (غيرها كما تريد)
```

## 🐛 استكشاف الأخطاء

### المشكلة: لا تعمل معالجة الملفات

**الحل:**
- تحقق من أن المتصفح يدعم JSZip
- تأكد من أن ملف PowerPoint بصيغة صحيحة (.pptx)

### المشكلة: لا تعمل Gemini API

**الحل:**
- تحقق من صحة المفتاح
- تأكد من اتصالك بالإنترنت
- تحقق من حدود الاستخدام المجاني

### المشكلة: فقدان البيانات

**الحل:**
- localStorage قد يتم مسحه إذا مسحت بيانات المتصفح
- عمّل نسخ احتياطية من وقت لآخر

## 📊 نسخ البيانات

لنسخ النتائج احفظ هذا في وحدة التحكم:

```javascript
// نسخ الأسئلة
JSON.stringify(JSON.parse(localStorage.getItem('quizQuestions')))

// نسخ النتائج
JSON.stringify(JSON.parse(localStorage.getItem('studentResults')))
```
